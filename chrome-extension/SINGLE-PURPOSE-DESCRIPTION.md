## Alternative: Detailed Version (If You Want to Be More Specific)

```
This extension has a single, narrow purpose: to audit web pages for AI search engine optimization (AEO) and general engine optimization (GEO) readiness. It specifically checks whether web pages are properly optimized for discovery and indexing by AI-powered search platforms such as ChatGPT (OpenAI), Claude (Anthropic), and Perplexity.

The extension performs 21+ automated checks including: meta title and description validation, structured data (JSON-LD) verification, Open Graph and Twitter Card tags, LLM.txt file existence, robots.txt AI crawler configuration (GPTBot, Claude-Web, PerplexityBot), XML sitemap validation, canonical URL checks, content quality analysis, FAQ section detection, internal/external linking analysis, schema validation, page speed indicators, accessibility checks, and mobile-friendliness.

The extension provides users with an overall AI readiness score (0-100), detailed pass/fail results for each check, and actionable recommendations for improvement. Users can export comprehensive PDF reports of audit results. All analysis is performed locally in the user's browser with no data collection, storage, or transmission to external servers.
```

## What Makes This a "Single Purpose"

1. **Narrow Focus**: The extension does ONE thing - audits pages for AI/SEO readiness. It doesn't try to do multiple unrelated things.

2. **Clear Scope**: It's specifically about checking optimization for AI platforms, not general web development or multiple SEO tools combined.

3. **Unified Functionality**: All 21+ checks serve the single purpose of determining AI readiness. They're not separate features but components of one audit process.

4. **No Feature Creep**: The extension doesn't try to be a website builder, content editor, analytics tool, or anything else - just an audit tool.

5. **Export Feature Aligns**: PDF export is part of the audit workflow, not a separate purpose - it helps users act on audit results.
